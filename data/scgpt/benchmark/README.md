# Clifti-GPT benchmark AnnData

Prepared **reference** and **query** `.h5ad` files for Clifti-GPT experiments live in this directory. They are not tracked in git.

## Download and extract

1. Download the benchmark archives from Zenodo:
   - **DOI:** [10.5281/zenodo.20491148](https://doi.org/10.5281/zenodo.20491148)
   - **Record:** https://zenodo.org/records/20491148
2. Extract so each cohort appears as a subfolder here, for example:

```
data/scgpt/benchmark/
├── ms/
│   ├── reference_annot.h5ad
│   └── query_annot.h5ad
├── hp5/
│   ├── reference.h5ad
│   └── query.h5ad
└── …
```

3. Download per-dataset **init weights** into [`models/init/`](../../../models/init/README.md): [10.5281/zenodo.20489646](https://doi.org/10.5281/zenodo.20489646).
4. Download the **scGPT whole-human checkpoint** into `models/pretrained_models/scGPT_human/` — see [scGPT on GitHub (MIT)](https://github.com/bowang-lab/scGPT) and cite [Cui et al., *Nature Methods* (2024)](https://www.nature.com/articles/s41592-024-02201-0). scGPT weights are not included in this repository; see [`../../../NOTICE`](../../../NOTICE).
5. **Myeloid scalability (required):** after extracting `myeloid.zip`, generate Top5–Top30 splits before running `MYELOID-top5` … `MYELOID-top30` experiments:

```bash
python data/myeloid_prep.py
```

This creates `myeloid-top5/` … `myeloid-top30/` from `myeloid/reference_adata.h5ad` and `myeloid/query_adata.h5ad`.

6. **COVID batch correction (required for `COVID-corrected`):** `covid.zip` includes `Covid.h5ad` plus uncorrected ref/query. After extract, activate the `batch_correction` conda environment and run:

```bash
conda activate batch_correction
bash data/correct_batch_effect.sh data/scgpt/benchmark
```

This writes `covid-corrected/`. It does **not** rebuild from `reference-raw.h5ad` / `query-raw.h5ad` alone.

## Cohorts

| Paper name | Folder | Reference | Query | Federated client column |
|------------|--------|-----------|-------|-------------------------|
| MS | `ms/` | `reference_annot.h5ad` | `query_annot.h5ad` | `split_label` |
| HP5 | `hp5/` | `reference.h5ad` | `query.h5ad` | `batch_name` |
| LUNG | `lung/` | `reference_annot.h5ad` | `query_annot.h5ad` | `sample` |
| CellLine | `cl/` | `reference.h5ad` | `query.h5ad` | `batch` |
| COVID | `covid/` | `reference-raw.h5ad` | `query-raw.h5ad` | `batch_group` |
| COVID (central corrected) | `covid-corrected/` | `reference_corrected.h5ad` | `query_corrected.h5ad` | `batch_group` | Generated by [`correct_batch_effect.sh`](../../correct_batch_effect.sh) |
| Myeloid (base) | `myeloid/` | `reference_adata.h5ad` | `query_adata.h5ad` | `batch` (source); see below for scaling |
| Myeloid Top5–Top30 | `myeloid-top5/` … `myeloid-top30/` | `reference.h5ad` | `query.h5ad` | `combined_batch` | Generated by [`myeloid_prep.py`](../../myeloid_prep.py) |

Registry used by experiment scripts: [`experiments/configs.sh`](../../../experiments/configs.sh).

## Optional: build from raw sources

Prep scripts in [`data/`](../../) can regenerate these splits from upstream cohort files — see the parent [`data/README.md`](../../README.md).
